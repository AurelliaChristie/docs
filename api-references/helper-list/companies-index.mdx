---
title: "Companies by Index"
api: "GET /index/{index}"
description: "Return list of stock tickers that belong to certain stock index"
---

<ResponseExample>
```json 200
["ACES.JK", "ADRO.JK", "AKRA.JK", "UNVR.JK"]
```

```json 429
{ "code": 429, "message": "Rate limit exceeded. Consider upgrading." }
```

</ResponseExample>

<Note>Available indices: `ftse`, `idx30`, `idxbumn20`, `idxesgl`, `idxg30`, `idxhidiv20`, `idxq30`, `idxv30`, `jii70`, `kompas100`, `lq45`, `sminfra18`, `srikehati`</Note>

### Header

<ParamField header="Authorization" type="string" required>
  Authorization header that should be filled with your Sectors API key.
</ParamField>

### Path

<ParamField path="index" type="string" required>
  The index name from which the list of companies is to be retrieved. For
  example: `ftse`, `idx30`, `idxbumn20`, `idxesgl`, `idxg30`, `idxhidiv20`, `idxq30`, `idxv30`, `jii70`, `kompas100`, `lq45`, `sminfra18`, `srikehati`.
</ParamField>

### Response

<ResponseField name="payload" type="string[]" required>
  Array of tickers of the given index.
</ResponseField>
