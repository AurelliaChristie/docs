---
title: "Companies by Subindustry"
api: "GET /companies/"
description: "Return list of companies from a subindustry"
---

### Header

<ParamField header="Authorization" type="string" required>
  Authorization header that should be filled with your Sectors API key.
</ParamField>

### Query Parameters

<ParamField query="sub_industry" type="string" initialValue="timber" required>
  Subindustry from which the list of companies is to be retrieved.
  <AccordionGroup>
    <Accordion title="Get the available sub_industry">
      Get the available `sub_industry` from the [Subindustries](./subindustries)
      endpoint.
    </Accordion>
    <Accordion title="Examples of the sub_industry">
      `business-support-services`, `electric-utilities`,
      `heavy-constructions-civil-engineering`, `road-transportation`, `timber`
    </Accordion>
    <Accordion title="Format">
      The API endpoint only accepts `sub_industry` in the **kebab case** format
      (lowercase and separated by "-").
    </Accordion>
  </AccordionGroup>
</ParamField>

### Response

<ResponseField name="payload" type="string[]">
  Array of ticker of companies in given subindustry
</ResponseField>

### Usage

This list can be used for the following endpoint:

- [Company Report](../report/company)

<ResponseExample>
```json 200
[
  "BALI.JK",
  "BTEL.JK",
  "CENT.JK",
]
```

```json 400
[
  "The requested sub_industry does not exist."
]
```

```json 429
{"code": 429, "message": "Rate limit exceeded. Consider upgrading."}
```
</ResponseExample>
