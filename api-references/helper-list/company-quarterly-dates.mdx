---
title: "Quarterly Financial Dates of a Company"
api: "GET /company/get_quarterly_financial_dates/{ticker}/"
description: "Return available quarterly financial dates of a given ticker"
---

### Header

<ParamField header="Authorization" type="string" required>
  Authorization header that should be filled with your Sectors Financial API
  key.
</ParamField>

### Path

<ParamField path="ticker" type="string" required>
  Ticker of the company for which the report is to be retrieved.
  <AccordionGroup>
    <Accordion title="Get the available ticker">
      Get the available `ticker` from the [Companies by
      Subsector](../helper-list/companies-subsector) or the [Companies by
      Subindustry](../helper-list/companies-subindustry) endpoints.
    </Accordion>
    <Accordion title="Examples of the ticker">
      `ASII`, `BBCA`, `BBRI`, `BMRI`, `BYAN`
    </Accordion>
    <Accordion title="Format">
      A valid `ticker` consists of 4 letters, optionally followed by '.jk', and
      is **case-insensitive**.
    </Accordion>
  </AccordionGroup>
</ParamField>

### Response

<ResponseField name="<financial_year>" type="object">
  The financial year of the quarterly financial report.
  <Expandable title="properties">
    <ResponseField name="report_date" type="array">
      The available quarterly financial dates in the financial year.
    </ResponseField>
  </Expandable>
</ResponseField>
