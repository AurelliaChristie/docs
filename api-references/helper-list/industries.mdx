---
title: "Industries"
api: "GET /industries/"
description: "Return list of all available industries"
---

### Header

<ParamField header="Authorization" type="string" required>
  Authorization header that should be filled with your Sectors API key.
</ParamField>

### Response

<ResponseField name="payload" type="Industry[]">
  <Expandable title="properties">
    <ResponseField name="Industry.subsector" type="string">
      A specific classification within a sector
    </ResponseField>

    <ResponseField name="Industry.industry" type="string">
      A broad classification of companies in similar economic activities
    </ResponseField>
  </Expandable>
</ResponseField>

### Usage

This list can be used for the following endpoint:

- [Companies by Subindustry](./companies-subindustry)

<ResponseExample>
```json 200
[
  {
    "subsector": "oil-gas-coal",
    "industry": "oil-gas"
  }
]
```

```json 429
{"code": 429, "message": "Rate limit exceeded. Consider upgrading."}
```
</ResponseExample>
