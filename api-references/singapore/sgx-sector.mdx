---
title: "SGX Sectors"
api: "GET /sgx/sectors/"
description: "Return list of all available sectors in SGX Index"
---

<ResponseExample>
```json 200
[
  "automobiles-auto-parts",
  "personal-household-products-services",
  ...
]
```

```json 429
{ "code": 429, "message": "Rate limit exceeded. Consider upgrading." }
```

</ResponseExample>

### Header

<ParamField header="Authorization" type="string" required>
  Authorization header that should be filled with your Sectors Financial API
  key.
</ParamField>

### Response

<ResponseField name="payload" type="Sector[]">
  <Expandable title="properties">
    <ResponseField name="Sector" type="array">
      A broad classification of companies in similar economic areas.
    </ResponseField>
  </Expandable>
</ResponseField>

### Usage

This list can be used for the following endpoints:

- [SGX Companies by Sector](./sgx-companies-by-sector)
- [SGX Top Companies by Sector](./sgx-top-companies)
