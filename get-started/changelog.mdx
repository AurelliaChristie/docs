## 2025-01-03

- Added new endpoints to get list of companies in SGX and KLSE Index from a sector

  - [https://api.sectors.app/v1/sgx/companies/?sector=\{sector}](../api-references/singapore/sgx-companies-by-sector)
  - [https://api.sectors.app/v1/klse/companies/?sector=\{sector}](../api-references/malaysia/klse-companies-by-sector)

- Added new endpoint to get quarterly financials metrics of a given ticker
  - [https://api.sectors.app/v1/financials/quarterly/\{ticker}/](../api-references/report/company-quarterly-financials)

## 2024-07-31

- Added new query parameters to support finding companies by industries

  - [https://api.sectors.app/v1/companies/?sub_industry=\{sub_industry}](../api-references/helper-list/companies-subindustry)

- Added new endpoint to support finding companies by index

  - [https://api.sectors.app/v1/index/\{index}/](../api-references/helper-list/companies-index)

- Added new endpoint to support finding companies with revenue and cost segments

  - [https://api.sectors.app/v1/companies/list_companies_with_segments/](../api-references/helper-list/companies-segment-list)

- Added `company_name` to the return value of most API endpoints

  - [https://api.sectors.app/v1/companies/](../api-references/helper-list/companies-subsector)
  - [https://api.sectors.app/v1/companies/top/](../api-references/ranking/top-tickers)
  - [https://api.sectors.app/v1/most-traded/](../api-references/ranking/most-traded)
  - These endpoints now return `company_name` in addition to `symbol`

- Change `sector/report` to `subsector/report`

  - [https://api.sectors.app/v1/subsector/report/\{sub_sector}/](../api-references/report/subsector)
